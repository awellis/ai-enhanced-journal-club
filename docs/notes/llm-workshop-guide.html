Using LLMs to Read Scientific Papers

A 90-Minute Interactive Workshop Guide

Workshop Overview

  • Duration: 90 minutes
  • Participants: ~20 people
  • Objective: Learn effective strategies for using LLMs to assist in reading and understanding scientific papers
  • Materials Needed:
    • Sample scientific papers (abstracts, methods, figures)
    • Access to LLM platform
    • Slides/projector for demonstrations
    • Timer for activities
    • Optional: Small prizes for winning teams

Detailed Activity Schedule

1. Opening Warm-up (10 minutes)

Activity: “Paper Pain Points” - Format: Pair discussion + group sharing - Instructions: 1. Pairs discuss (3 min): “Share your worst experience trying to understand a scientific paper” 2. Quick-fire round (5 min): Each pair shares their most memorable story 3. Facilitator notes common themes (2 min) - Purpose: Build rapport, identify shared challenges, set context for LLM solutions

2. Interactive Demo (15 minutes)

Activity: “LLM Paper Whisperer” - Setup: Sample Abstract: Recent advances in transformer architectures have revolutionized natural language processing... [Include complex, jargon-heavy abstract] - Process: 1. Display abstract (2 min) 2. Audience suggests questions (5 min) 3. Live LLM interaction (5 min) 4. Discussion of effective vs. ineffective queries (3 min)

3. “Abstract Detective” Game (20 minutes)

Setup: Create 4-5 teams - Materials per team: - Same scientific abstract - Different LLM chat transcripts (some good, some problematic) - Evaluation rubric - Scoring Criteria: - Identification of effective techniques (+2 points) - Spotting potential pitfalls (+2 points) - Creative improvement suggestions (+1 point) - Timeline: - Analysis (10 min) - Team presentations (8 min) - Winner announcement (2 min)

4. “Build-A-Prompt” Workshop (15 minutes)

Challenge Structure: - Phase 1: Prompt Creation (7 min) - Teams craft prompts following best practices: 1. Clear context setting 2. Specific instructions 3. Output format requirements 4. Follow-up guidance - Phase 2: Testing (5 min) - Live testing of top prompts - Phase 3: Analysis (3 min) - Group discussion of effective elements

5. “Figure it Out” Challenge (15 minutes)

Activity Structure: - Provided Materials: - Complex scientific figure with caption - Template for analysis: 1. Simple Explanation: 2. Key Limitations: 3. Follow-up Questions: 4. LLM Prompt Used: - Team Tasks: 1. Create effective prompt (4 min) 2. Generate explanation (4 min) 3. Cross-team review (4 min) 4. Best practices discussion (3 min)

6. “Peer Review Partner” Exercise (10 minutes)

Individual Activity: - Steps: 1. Read provided conclusion section (2 min) 2. Use LLM to analyze: - Citation verification - Logic checking - Gap identification 3. Compare findings with neighbor (3 min) 4. Group discussion of insights (5 min)

7. Closing Activity (5 minutes)

“Commitment Cards”: - Each participant writes: 1. One technique they’ll implement 2. One potential challenge they anticipate 3. How they’ll measure success

Best Practices for Facilitation

Engagement Tips

  1. Energy Management:
    • Start with high energy
    • Mix up sitting/standing activities
    • Use movement during transitions
  2. Time Management:
    • Set clear timers
    • Have “time buffer” activities ready
    • Be flexible with discussion times
  3. Participation Strategies:
    • Use round-robin for sharing
    • Encourage quiet participants
    • Manage dominant voices

Technical Considerations

  1. LLM Platform:
    • Test connectivity beforehand
    • Have backup examples ready
    • Prepare for common error scenarios
  2. Content Preparation:
    • Use diverse paper examples
    • Prepare intentionally flawed prompts
    • Include multiple scientific disciplines

Success Metrics

  • Participant engagement levels
  • Quality of generated prompts
  • Post-workshop feedback
  • Implementation commitments
  • Follow-up participation

Note: Adjust timing and activities based on participant background and engagement levels. Have backup activities ready for flexibility.
